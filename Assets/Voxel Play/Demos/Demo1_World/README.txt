Welcome to Voxel Play!

This demo scene contains a ready-to-play world with different biomes, voxels, sounds and textures.
Because there's so much content in this sample world, all resources are contained within VoxelPlay/Resources/Worlds folder so you can reuse the same world for your game and can remove the demo folders when you no longer need them.

This demo scene is a blueprint for a voxel modeler.

If provides:
- Navigation: orbit + free move as you approach the model (automatic switch)
- Voxel placement (right mouse button)
- Voxel coloring (left mouse button)
- Voxel removal (press X to toggle draw/erase mode)
- Load/Save to temporary string format so you can store it anywhere (press F2/F3 to save/load)
- Import Qubicle binary format (use the Voxel Play Environment tools section in its inspector to pick up a .qb file and generate the Voxel Play Model Definition then assign it to colorizer gameobject to load it)
